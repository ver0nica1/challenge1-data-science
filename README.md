# Challenge 1 - Data Science

Análisis exploratorio de 4 tiendas para recomendar cuál debería vender el Sr. Juan.

## Contenido
- `store.ipynb`: notebook principal con todo el análisis.
- `base-de-datos-challenge1/`: archivos CSV de las tiendas.

## Qué se analiza
1. Facturación total por tienda
2. Ventas por categoría
3. Calificación promedio
4. Productos más y menos vendidos
5. Costo de envío promedio

## Tecnologías
- Python 3
- pandas
- matplotlib
- Jupyter Notebook

## Cómo ejecutar
1. Abre una terminal en esta carpeta.
2. Instala dependencias:
   ```bash
   pip install pandas matplotlib notebook
   ```
3. Inicia Jupyter:
   ```bash
   jupyter notebook
   ```
4. Abre `store.ipynb` y ejecuta las celdas en orden.

## Resultado
El informe final del notebook recomienda vender la **Tienda 4**, basada en el análisis comparativo de desempeño.
